# QuantumAnimalChess

たのしい量子どうぶつしょうぎです。コンピューターと対戦できます。

## ファイル

* QuantumAnimalChess.exe : ゲーム自体 (.NET Framework 4.0 がインストールされたWindowで動作します)
* /src : ソースコード (Visual Studio 2012を使ってC#で書いています)
